Project Architecture Overview

WorkFlow is built as a mobile-first responsive web application. It integrates both frontend interface screens and local API route endpoints inside a single Next.js project to allow zero-configuration full-stack deployment.

Tech Stack Element Details & Implementation
Framework Next.js 16 (App Router), React 19, TypeScript. Utilizes React Server Components (RSC) and client-side page transitions.
State Management Zustand with persist middleware to store active user roles (Employee, Manager, HR, Admin) and onboarding switches locally in the browser.
Styling Tailwind CSS (version 4) implementing a customized, premium Teal & Orange palette with frosted-glass backdrop filters.
API & DB Mock Layer TanStack React Query handles server state fetching and mutations. The server side runs API endpoints that read from and write to next.js/db.json on local server instances.

Directory Structure Map

The code is divided into Next.js App router folders, components, custom hooks, and shared libraries.

├── lib/                        # Common helper libraries & mock database wrappers
└── db.json                     # Server state mock database file

1. Onboarding Module

Handles the entire new hire experience with task phases, CEO message, team introductions, relocation support, and handoff.

Core Features & Details:

Feature Capability Technical Implementation
5-Phase Checklist Includes pre-joining, day-1, week-1, week-2, and month-1. Displays task details like priority levels (High/Medium/Low) and assignees. Restricts marking complete to employee-assigned tasks; others show "Pending action by [Assignee]". Writes date on completion.
Welcome Message Video CEO card triggers modal with image profile of CEO Arun Krishnamurthy. Play button triggers browser window.speechSynthesis API reading welcome text aloud, paired with custom play/pause status and audio visualizer.
Relocation Tracker Displays accommodation, visa status, travel details, local buddy details, allowances with currency formatting, and support tickets grid. Contains form to post relocation tickets directly to the mock database.
Meet Team Introductions Displays key contacts. "Mark as Met" button updates status in backend database dynamically.
Complete Handoff Flow Clicking "Complete Onboarding" calls completion API route, resets Zustand state, and moves the employee to the standard homepage.
